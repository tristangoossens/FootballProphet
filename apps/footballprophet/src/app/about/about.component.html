CASE (The case that i have designed for this project)

FootballProphet is an online platform where users can compete with their friends to see who can predict the outcomes of matches in various football leagues the best. The platform allows users to make their own predictions for matches in leagues such as the Premier League, La Liga, and Eredivisie By making a prediction, users can earn points based on the accuracy of their predictions. The platform is interactive, and users can compete against their friends in a variety of ways, such as creating private leagues to compete with friends and participating in public leagues. FootballProphet's leaderboard is updated in real time, allowing users to track their progress and see how they stack up against their friends. With FootballProphet, football fans can combine their love for the game with a healthy competition with their friends.

USE-CASES (Some of the use-cases that this website contains)

USER

  • US_U_01: As a user, I want to be able to create an account to participate in an existing league, so that I can compete with my friends to see who can predict football matches the best.
  • US_U_02: As a user, I want to be able to create a league that can be both "invite-only" and "public", so that I can compete with my friends or strangers to see who is the best football expert.
  • US_U_03: As a user, I want to be able to make a prediction for an upcoming match, in order to earn points and win the league.
  • US_U_04: As a user, I want to be able to see the current standings of the pool, so that I can see how I am doing compared to my friends.
  • US_U_05: As a user, I want to be able to look up information about a specific league, its teams and its fixtures so i can make better predictions.

POOL OWNER

  • US_PO_01: As a pool owner, I want to be able to create an invite link for my pool, so that I can invite friends to a private pool.
  • US_PO_02: As a pool owner, I want to be able to kick any unwanted users from my pool.

ADMINISTRATOR

  • US_A_01: As an administrator, I want to be able to create a new league, so that users can make pools for that league.
  • US_A_02: As an administrator, I want to be able to edit a league, so that i can change wrongfully entered data.
  • US_A_03: As an administrator, I want to be able to create teams and fixtures, so that user can make predictions for it.
  • US_A_04: As an administrator, I want to be able to score past predictions, so that users can see their current standings.
